# Auth Profile Project (Backend + Frontend)

This archive contains:
- auth-profile-backend: Node.js + Express + MongoDB backend
- auth-profile-frontend: React frontend

See `.env.example` in backend for env variables.

Run backend:
1. cd auth-profile-backend
2. npm install
3. create .env from .env.example and fill MONGO_URI and JWT_SECRET
4. npm run dev

Run frontend:
1. cd auth-profile-frontend
2. npm install
3. npm start
